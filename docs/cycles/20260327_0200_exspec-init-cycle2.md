## Implementation Notes

### Goal
`exspec init` コマンドで `.exspec.toml` をゼロ設定で生成できるようにする。Cycle 2 では言語検出 (Cycle 1) に加えてフレームワーク検出・custom_patterns 自動設定・CLI統合を完成させ、`exspec init` を end-to-end で動作可能な状態にする。

### Background
Cycle 1 で `detect_languages` + `generate_toml` の基盤を実装済み。元の Cycle 2 (フレームワーク検出) と Cycle 3 (CLI統合) を統合して1サイクルで完成させる。

### Design Approach

**フレームワーク検出 (`detect_frameworks`)**

言語検出後に呼び出し、依存ファイルの内容を読んでフレームワークを特定:

| フレームワーク | 検出方法 |
|--------------|---------|
| pytest | `conftest.py` 存在 OR `pyproject.toml` に "pytest" |
| Django | `manage.py` 存在 |
| Flask | `pyproject.toml`/`requirements.txt` に "flask" (case-insensitive) |
| FastAPI | 同上に "fastapi" |
| Jest | `package.json` devDependencies に "jest" |
| Vitest | `package.json` devDependencies に "vitest" |
| NestJS | `package.json` dependencies に "@nestjs/core" |
| PHPUnit | `composer.json` require-dev に "phpunit/phpunit" |
| Laravel | `composer.json` require に "laravel/framework" |
| Pest | `composer.json` require-dev に "pestphp/pest" |

**custom_patterns 生成 (`get_custom_patterns`)**

| フレームワーク | custom_patterns |
|--------------|----------------|
| Laravel | `["expects", "shouldReceive", "shouldBeCalled"]` |
| Pest | `["expect("]` |
| pytest | `["mock.assert_*"]` |
| NestJS | (不要 — expect は標準検出済み) |

**generate_toml 拡張**

`generate_toml(langs, frameworks)` に `frameworks` 引数を追加。custom_patterns がある場合 `[assertions]` セクションを出力に追加。

**CLI統合**

```rust
/// Initialize .exspec.toml with auto-detected settings
Init(InitArgs),

struct InitArgs {
    #[arg(default_value = ".")]
    path: String,
    #[arg(long)]
    dry_run: bool,
    #[arg(long)]
    force: bool,
}
```

`run_init` の振る舞い:
- 既存 `.exspec.toml` あり + `--force` なし → エラーメッセージ + exit 1
- `--dry-run` → stdout に出力のみ、ファイル書き込みなし
- 正常時: 検出結果を表示 + ファイル書き込み

**出力フォーマット例**

```
$ exspec init
Detected: Python (pytest, Flask)
Generated: .exspec.toml

$ exspec init --dry-run
Detected: Python (pytest, Flask)
# Generated by `exspec init`
# Detected: Python (pytest, Flask)
[general]
lang = ["python"]
[assertions]
custom_patterns = ["mock.assert_*"]
[paths]
ignore = [".venv", "__pycache__"]
```
